## 📦 About

- ✅ One-click pull all skills to local

## 🚀 Quick Start

### Method 1: One-click Sync (Recommended)

Execute in Hacker Terminal:

```
/skills sync
```

Automatically pull all latest skills from this repository.

### Method 2: Manual Download

1. Download this repository as ZIP
2. Extract skills from `skills/` directory
3. Place them in Hacker Terminal's `skills/` directory
4. Restart terminal or execute `/skills reload`

## 🛠️ Develop Your Own Skills

### Skill Format

Each skill is a directory containing a `SKILL.md` file that describes the skill's functionality, usage, and implementation.

### Skill Template

```
my-skill/
├── SKILL.md          # Skill description document
├── src/              # Source code (optional)
└── assets/           # Assets (optional)
```

### Submit Your Skill

1. Fork this repository
2. Create your skill directory
3. Write SKILL.md documentation
4. Submit Pull Request
5. Wait for review and merge
