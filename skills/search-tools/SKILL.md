# 搜索工具 (search-tools)

## 说明
提供文件内容搜索、正则搜索、全文检索等功能，快速定位文件和内容。

## 权限要求
Level 1 及以上

## 命令列表

### find - 文件名搜索
- **用法**：find <目录> <文件名模式>
- **说明**：在指定目录下搜索匹配的文件
- **示例**：
  ```
  find C:\projects *.txt
  find D:\docs report*.docx
  ```

### grep - 文件内容搜索
- **用法**：grep <目录> <关键词>
- **说明**：在指定目录下搜索包含关键词的文件
- **示例**：
  ```
  grep C:\projects "TODO"
  grep D:\docs "重要"
  ```

### grep-regex - 正则搜索
- **用法**：grep-regex <目录> <正则表达式>
- **说明**：使用正则表达式搜索文件内容
- **示例**：
  ```
  grep-regex C:\src "void\s+\w+\s*\("
  grep-regex D:\logs "\d{4}-\d{2}-\d{2}"
  ```

### replace - 批量替换
- **用法**：replace <目录> <查找内容> <替换内容>
- **说明**：批量替换文件中的文本内容
- **示例**：
  ```
  replace C:\projects "old" "new"
  replace D:\docs "v1.0" "v2.0"
  ```

### count - 统计出现次数
- **用法**：count <目录> <关键词>
- **说明**：统计关键词在文件中出现的次数
- **示例**：
  ```
  count C:\src "printf"
  count D:\docs "重要"
  ```

### file-type - 按类型搜索
- **用法**：file-type <目录> <扩展名>
- **说明**：搜索指定类型的文件
- **示例**：
  ```
  file-type C:\projects .c
  file-type D:\docs .pdf
  ```

### large-files - 大文件查找
- **用法**：large-files <目录> [大小阈值]
- **说明**：查找大于指定大小的文件（默认 100MB）
- **示例**：
  ```
  large-files C:\
  large-files D:\ 50MB
  ```

### duplicate - 重复文件查找
- **用法**：duplicate <目录>
- **说明**：查找目录中的重复文件（基于 MD5）
- **示例**：
  ```
  duplicate D:\photos
  ```

## 搜索选项
- **递归搜索**：默认递归搜索所有子目录
- **忽略大小写**：默认不忽略，可加 -i 参数
- **显示行号**：grep 命令默认显示行号
- **颜色高亮**：匹配结果高亮显示

## 性能说明
- 小目录（<1000 文件）：秒级完成
- 中目录（1万文件）：几秒到几十秒
- 大目录（10万+ 文件）：建议缩小搜索范围
- 建议搜索 SSD 上的目录，速度更快

## 注意事项
- 搜索系统目录可能需要管理员权限
- 正则表达式请确保语法正确
- 批量替换前建议先备份重要文件
